## Extraction Rules

1. Extract every entity you can identify — equipment, failure modes, incidents, technicians, compliance rules, work orders, and tacit knowledge.
2. For each entity, provide all available properties.
3. For each relationship between entities, specify both endpoints.
4. If a property is not mentioned, omit it entirely (do NOT use null or "unknown").
5. Use consistent naming — if the same entity appears multiple times, use the same name each time.
6. Dates should be in ISO 8601 format (YYYY-MM-DD) where possible.
