# Extraction Prompt Templates

This directory holds all Claude prompt templates, kept separate from
application logic so they can be iterated on quickly without touching code.

## Files (to be created in Phase 2+)

- `extraction.txt` — Structured entity extraction from documents
- `rag_answer.txt` — RAG answer generation with citations
- `confidence.txt` — Confidence tier classification
- `compliance_reasoning.txt` — Compliance gap reasoning trace
- `interview_questions.txt` — Tacit knowledge interview question generation
