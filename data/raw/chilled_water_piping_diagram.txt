FLOW PATH SUMMARY
================================================================================

PRIMARY LOOP (Chiller Side):
  Chiller Evaporator → Primary CHW Pump → Common Pipe → Return to Chiller
  
  Flow Rate:     ~570 GPM per chiller (at design ΔT)
  Pipe Size:     6" (supply and return headers)
  Pipe Material: Schedule 40 carbon steel, welded joints
  Insulation:    1.5" closed-cell foam with vapor barrier

SECONDARY LOOP (CRAH Side):
  Common Pipe → Secondary CHW Pump → Supply Header → CRAH Coils →
  Return Header → Common Pipe
  
  Flow Rate:     ~430 GPM total (variable, VSD controlled)
  Branch Size:   2.5" per CRAH unit
  Control Valve: 2-way modulating (Belimo, 0-10V actuator)
  ΔT Design:     14°F / 7°C (44°F supply, 58°F return)

CONDENSER LOOP (Heat Rejection):
  Chiller Condenser → CW Pump → Cooling Tower → Return to Condenser
  
  Flow Rate:     ~750 GPM per tower
  Pipe Size:     8" (supply and return)
  Design ΔT:     10°F (85°F supply, 95°F return)

MONITORING POINTS
================================================================================

Critical sensors for BMS (Building Management System) integration:

  CHILLED WATER SYSTEM:
    • CHW supply temperature (at header)
    • CHW return temperature (at header)
    • CHW differential pressure (across header)
    • CHW flow rate (at each pump)
    • Chiller status (run/stop/fault per chiller)
    • Chiller kW consumption
    
  CONDENSER WATER SYSTEM:
    • CW supply temperature
    • CW return temperature
    • Cooling tower fan speed
    • Cooling tower basin level
    • Outdoor wet-bulb temperature
    
  CRAH UNITS (per unit):
    • Supply air temperature
    • Return air temperature
    • Supply air humidity
    • Fan speed (%)
    • Chilled water valve position (%)
    • Filter differential pressure
    • Unit status (run/stop/fault)
    • Power consumption (kW)
    
  ROOM CONDITIONS:
    • Cold aisle temperature (per row, at 3 heights)
    • Hot aisle temperature (per row)
    • Under-floor pressure
    • Room humidity (multiple points)

================================================================================
MAINTENANCE REQUIREMENTS (PIPING SYSTEM)
================================================================================

MONTHLY:
  □ Check glycol concentration (if applicable)
  □ Verify system pressure (normal: 15-30 PSI)
  □ Inspect expansion tank pressure (pre-charge: 12 PSI)
  □ Check chemical treatment levels
  □ Inspect for leaks at valve stems and pump seals

QUARTERLY:
  □ Clean strainers
  □ Check valve operation (isolation and control valves)
  □ Verify flow rates at each CRAH branch
  □ Inspect pipe insulation for damage/condensation
  □ Test safety relief valves
  □ Flush condensate drains

ANNUALLY:
  □ Full water quality analysis (pH, TDS, bacteria, corrosion coupons)
  □ Vibration analysis on all pumps
  □ Thermographic survey of all piping connections
  □ Recalibrate all temperature and pressure sensors
  □ Inspect cooling tower fill media and basin
  □ Clean and inspect all heat exchangers
